# Student Management System

A simple full-stack Student Management System built using FastAPI, React.js, and MySQL.  
The application allows users to add, view, and delete student records through a clean frontend interface connected to REST APIs.

---

## Tech Stack

### Frontend
- React.js
- Fetch API

### Backend
- FastAPI
- Python
- SQLAlchemy

### Database
- MySQL

---

## Features

- Add new students
- View all students
- Delete student records
- REST API integration
- Real-time frontend updates
- MySQL database connectivity
